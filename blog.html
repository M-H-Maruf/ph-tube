What is the scope of var, let, and const?

Var keyword declares a variable which is scoped to its current execution context, optionally initializing to a value. Let keyword declares a block scoped variable, optionally initializing it to a value. Const keyword declares constants which are block scoped much like variables defined using let but the value of a constant cannot change. The const declaration creates a read-only reference to a value.

What is the use cases of null and undefined?

'null' is used to represent an intentional absence of a value or to clear a variable, indicating emptiness. 'undefined' indicates that a variable has been declared but hasn't been assigned a value yet, or it's used as the default value for missing function arguments or nonexistent object properties.

What is the meaning of REST API?

A REST API (also known as RESTful API) is an application programming interface (API or web API) that conforms to the constraints of REST architectural style and allows for interaction with RESTful web services. REST stands for representational state transfer and was created by computer scientist Roy Fielding.
